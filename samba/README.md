#家庭助理附加组件:桑巴共享

使用Windows文件共享通过网络共享您的配置。

![支持aarch64架构][aarch64-shield] ![支持amd64架构][amd64-shield] ![支持armhf架构][armhf-shield] ![支持armv7架构][armv7-shield] ![支持i386架构][i386-盾]

##关于

此附加组件允许您通过网络在不同的操作系统之间实现文件共享。
它允许你用Windows和macOS设备访问你的配置文件。
